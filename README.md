# Agora 1v1 Video Call

A simple, real-time 1v1 video communication application built using the latest Agora Web SDK (v4).

## Features
- **Real-time Video/Audio**: High-quality 1v1 communication.
- **Media Controls**: Toggle your microphone and camera on/off during the call.
- **Responsive UI**: Works on both desktop and mobile browsers.
- **Easy Setup**: Just enter your Agora App ID and channel name to start.

## How to Use
1. **App ID**: Get your App ID from the [Agora Console](https://console.agora.io/).
2. **Channel Name**: Choose any name for your meeting room.
3. **Token**: If your project has security enabled, provide a temporary token.
4. **Join**: Share the channel name with a friend to start the call!

## Technologies Used
- [Agora Web SDK v4](https://docs.agora.io/en/video-calling/get-started/get-started-sdk?platform=web)
- HTML5 / CSS3 / Vanilla JavaScript
- Font Awesome for icons
